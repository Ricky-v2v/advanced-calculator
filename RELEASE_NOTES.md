# 发布说明

## v1.0.0 (2025-02-15)

### 新功能
- 命令行界面 (CLI) 支持
- 图形用户界面 (GUI) 支持
- 基本运算和科学函数计算
- 复数运算支持
- 单位转换功能
- 历史记录管理
- 多语言支持
- 自动更新检查

### 系统要求
- Python 3.8+
- 支持的操作系统：Windows, macOS, Linux

### 安装包
- Windows: 
  - calculator-cli-windows.exe
  - calculator-gui-windows.exe
- macOS:
  - calculator-cli-darwin
  - calculator-gui-darwin
- Linux:
  - calculator-cli-linux
  - calculator-gui-linux
